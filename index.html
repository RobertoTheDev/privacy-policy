Política de Privacidade

O EduBiotec é um projeto voltado para a educação e tem como compromisso respeitar a privacidade dos usuários.

Coleta de dados

Este aplicativo não coleta dados pessoais identificáveis, como nome, e-mail ou senha.

Uso de informações

Como não há coleta de dados pessoais, nenhuma informação é utilizada, armazenada ou compartilhada.

Compartilhamento de dados

Não compartilhamos qualquer tipo de dado com terceiros.

Segurança

Mesmo sem coleta de dados pessoais, adotamos boas práticas para garantir a segurança e o bom funcionamento do aplicativo.

Alterações nesta política

Esta política pode ser atualizada periodicamente. Recomendamos a revisão desta página para se manter informado.
